WOA

自從 Netscape 被黑暗料理界打倒之後，WebTop 的概念就漸漸的被人們所淡忘， 雖然正義假面 Google，其實仍然默默的在打造秘密武器....

一直到 O'Reilly 大師，向鄉民們開示了 Web 2.0 的概念之後，突然之間 正義的味方又開始鼓譟起了起來，同心協力的將輕量級的武器 HTTP，RSS，REST，AJAX 一同合體(mashup)成為了 Web-Oriented Architecture (WOA) 聖戰士

在併購與上市的金錢遊戲之中，我們要從技術的觀點出發，看看如何應用開放源碼軟體 來架構您的 Web 2.0 網站

WOA = SOA + Web + REST (Gartner, 2005)

Henri Chen

ZK, Ajax web framework with java

ZK 是一套以 AJAX/XUL/Java 為基礎的網頁應用程式開發框架，在設計 AJAX 網路應用程式時，輕鬆簡便的操作就像設計桌面程式一樣。 ZK 包含了一個以 AJAX 為基礎、事件驅動（event-driven）、高互動性的引擎，同時還提供了多樣豐富、可重複使用的 XUL 與 HTML 組件，以及以 XML 為基礎的使用介面設計語言 ZK User-interfaces Markup Language (ZUML)。ZK 的開發者 Henri Chen 將親自向各位介紹這套開發框架。
